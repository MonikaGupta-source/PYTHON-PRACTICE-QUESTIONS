# 🐍 Python Practice – Logic Building

> A no‑frills collection of Python practice questions.  
> Each file contains **multiple problems** – covering loops, conditionals, lists, strings, functions, patterns, and more.  
> The goal: **consistent daily practice** to sharpen logic and problem‑solving skills.

---

## 📁 Repository Structure

> 💡 *Why `.vscode/`?* – That’s just where I keep my practice scripts. No special meaning.

---

## 📚 What’s Inside Each File?

Each `LOGIC-BUILDING-XX.py` file contains **10–20 independent problems**, such as:

- **Conditionals** – even/odd, leap year, grade calculator  
- **Loops** – factorial, Fibonacci, prime check, sum of digits  
- **Lists** – max/min, reverse, remove duplicates, list comprehension  
- **Strings** – palindrome, anagram, vowel count, reverse words  
- **Dictionaries** – word frequency, character counter  
- **Number theory** – GCD, LCM, Armstrong, perfect number  
- **Patterns** – stars, pyramids, number triangles  
- **Basic algorithms** – linear search, bubble sort, binary search  
- **Mini challenges** – FizzBuzz, guess the number, simple ATM simulation  

Problems are **not grouped by topic** – they are shuffled to mimic real coding challenges.
